# Add HealthController To Connect With UptimeRobot

## Create Health Controller

### File: HealthController.cs
```csharp
using Microsoft.AspNetCore.Mvc;

namespace mktsystem.Controllers;

[ApiController]
[Route("api/health")]
public class HealthController : ControllerBase
{
    // Simple GET endpoint for uptime monitoring
    [HttpGet]
    public IActionResult Get()
    {
        return Ok("Healthy");
    }
}
```

### ✅ Notes

1. URL to ping:

```yaml
https://mkt-system.onrender.com/api/health
```

2.	Lightweight:

- DB calls, no auth → super safe for frequent pings (every 3–5 min)

3.	UptimeRobot setup:

- Monitor type: HTTP(s)
- Interval: 5 min
- URL: https://mkt-system.onrender.com/api/health

4. Optional:

- Add headers or logging if you want to track uptime requests, e.g., log each ping.
